	pip list

	cat /goinfre/$USER/miniconda3/envs/42AI-irifarac/lib/python3.7/site-packages/numpy-1.21.5.dist-info/METADATA
Elimina el paquete numpy.
	pip uninstall numpy

	pip install numpy
